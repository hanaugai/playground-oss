
- gomod

create go app under ./gomod/main.go with non-latest library version.

```
$ cd gomod
$ go mod init app
$ go get github.com/mattn/go-runewidth@v0.0.10
$ go mod tidy
```

  --env "PACKAGE_MANAGER=go_modules" \
  --env "DIRECTORY_PATH=/dependabot/gomod" \
